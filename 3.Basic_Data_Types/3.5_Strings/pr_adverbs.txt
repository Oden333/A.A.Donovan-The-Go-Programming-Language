| Тип данных      | Формат    | Флаги (adverbs)              | Примеры использования           | Описание                                                                |
|-----------------|-----------|------------------------------|---------------------------------|-------------------------------------------------------------------------|
| Общие           | %v        | + #  0 - *                   | %+v, %#v, % 5v, %-10v           | Универсальный формат (выводит значение)                                 |
|                 | %T        | -                            | %T                              | Тип переменной (не значение)                                            |
| Логические      | %t        |   0 - *                      | %t, %05t, %-5t                  | true/false                                                              |
| Целые числа     | %b        | +   0 - *                    | %+b, %08b                       | Двоичный формат (например, 11011)                                       |
|                 | %d        | +   0 - *                    | %+d, %05d                       | Десятичный формат (42)                                                  |
|                 | %o        | +   0 - * #                  | %#o, %+05o                      | Восьмеричный (с # добавляет 0)                                          |
|                 | %x/%X     | +   0 - * #                  | %#x, %08X                       | Шестнадцатеричный (с # добавляет 0x)                                    |
|                 | %c        |   0 - *                      | %c, %5c                         | Символ Unicode                                                          |
|                 | %U        | +   #                        | %#U, %+U                        | Unicode-код (например, U+03C0)                                          |
| Строки          | %s        |   0 - * .                    | %s, %-10.5s                     | Строка (с обрезкой до 5 символов)                                       |
|                 | %q        |   0 - * .                    | %q, %10q                        | Строка в кавычках с экранированием                                      |
|                 | %x/%X     |   0 - *                      | %x, %10X                        | HEX-дамп строки                                                         |
| Дробные числа   | %f        | +   0 - * .                  | %+.2f, %-08.3f                  | Десятичная дробь (3.141592 → 3.14)                                      |
|                 | %e/%E     | +   0 - * .                  | %+.2e, %10E                     | Экспоненциальная запись (3.14e+00)                                      |
|                 | %g/%G     | +   0 - * .                  | %g, %+.5G                       | Автовыбор (%f или %e)                                                   |
| Указатели       | %p        |   0 - * #                    | %p, %#p                         | Адрес указателя (0x7ffeefbff)                                           |
| Специальные     | %%        | -                            | %%                              | Выводит знак %                                                          |

------------------
### Ключевые флаги (adverbs):
1.  +     : Всегда показывать знак (+/-)
    Пример: %+d → "+42"

2.  ' '   : Пробел вместо + для положительных чисел
    Пример: % d → " 42"

3.  0     : Заполнение нулями слева
    Пример: %05d → "00042"

4.  -     : Выравнивание влево
    Пример: %-10s → "Hello     "

5.  *     : Динамическая ширина/точность (из аргумента)
    Пример: %*.*f, 5, 2, 3.1415 → " 3.14"

6.  #     : Альтернативный формат:
    - %#x → "0x2a"
    - %#o → "052"
    - %#v → Значение в синтаксисе Go

7.  .N    : Точность (для чисел — знаки после точки, для строк — обрезка)
    Пример: %.2f → "3.14", %.3s → "Hel"
